> **ActionCalledFromServerError**: Se llamó a una acción desde una página de servidor o un endpoint del servidor sin usar `Astro.callAction()`. Este wrapper debe utilizarse para llamar acciones desde el código del servidor.

## ¿Qué salió mal?
La acción llamada desde una página de servidor o endpoint sin usar `Astro.callAction()`.
